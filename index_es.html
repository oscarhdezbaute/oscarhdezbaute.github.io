SteelStudio

SteelStudio es una aplicación web desarrollada con tecnologia .NET, patrón ASP.NET MVC y Entity framework Code First.
Esta aplicación web gestiona toda la información sobre los costos de materiales, categorias de productos, diseños de productos de una compañia que trabaja con acero inoxidable.

SISTEMA DE CONTROL PARA PESAJE DE VEHICULOS

El sistema de control, pesaje y facturación electrónica para camiones es una aplicación desktop desarrollada con tecnologia .NET, C#, y Microsoft SQL Server.
Este sistema trabaja simultaneamente con varios indicadores de pesaje, sensores infrarrojos, cámaras IP, barreras o plumas, y sirenas. En la actualidad esta aplicación está trabajando en varias fábricas en Cuba.
Este sistema fue desarrollado bajo acuerdo de confidencialidad NDA (non-disclosure agreement)

Sistema de Gestión
TDL

El sistema de gestión de TDL es una aplicación web desarrollada con tecnologia .NET, ASP.NET Webforms y Microsoft SQL Server.
TDL es una empresa metalúrgica argentina, esta ofrece soluciones a varias ramas del mercado argentino. Esta aplicación web gestiona toda la información referente a la administración, ventas, ingenieria, producción, calidad, logística y el departamento de compras.
Este sistema fue desarrollado bajo acuerdo de confidencialidad NDA (non-disclosure agreement)
